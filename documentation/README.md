## Documentation

This folder documents each module by functionality so you can reuse pieces without
running the full pipeline.

Index:
- `documentation/catalog.md`: build the sensor catalog JSONL from YANG.
- `documentation/ingestion.md`: embed and ingest into Qdrant.
- `documentation/retrieval.md`: retrieve chunks from Qdrant (no RAG).
- `documentation/rag.md`: RAG wiring, inputs, and outputs.
- `documentation/yang.md`: import IOS XR YANG modules via sparse checkout.
